# 快速开始 - 零配置运行

## 超简单！3秒启动

### Windows 用户
1. 解压下载的压缩包
2. **双击 `双击启动.bat`**
3. 等待浏览器自动打开，完成！

### Linux/Mac 用户
1. 解压下载的压缩包
2. 在终端中运行：
   ```bash
   chmod +x 双击启动.sh
   ./双击启动.sh
   ```
3. 等待浏览器自动打开，完成！

---

## 就这么简单！

脚本会自动完成以下所有操作：
- ✅ 检测并安装 Node.js（如果没有）
- ✅ 安装所有项目依赖
- ✅ 启动后端服务（8899端口）
- ✅ 启动前端服务（8080端口）
- ✅ 自动打开浏览器

---

## 停止服务

### Windows
双击运行 `stop.bat`

### Linux/Mac
```bash
./stop.sh
```

---

## 注意事项

### 首次运行
- 首次运行需要下载依赖，可能需要 **3-5分钟**
- 请确保网络连接正常
- 国内用户如果下载慢，可以参考下方配置npm镜像

### 运行中
- **不要关闭**弹出的命令行窗口（Windows）
- 等待约 **40-60秒** 前端编译完成
- 看到 "Compiled successfully" 后即可使用

### 端口占用
如果提示端口被占用：
- Windows: 运行 `stop.bat` 后重新启动
- Linux/Mac: 运行 `./stop.sh` 后重新启动

---

## 可选优化（针对国内用户）

### 配置npm镜像加速
如果下载依赖很慢，可以配置淘宝镜像：

**Windows (在 cmd 中运行):**
```cmd
npm config set registry https://registry.npmmirror.com
```

**Linux/Mac (在终端中运行):**
```bash
npm config set registry https://registry.npmmirror.com
```

配置完成后，重新运行启动脚本即可。

## 自动安装 Node.js 说明

### Windows
- 优先使用 **winget**（Windows 10/11自带）
- 备选使用 **Chocolatey**
- 如果都不可用，会自动打开 Node.js 官网引导手动安装

### Linux
- **Ubuntu/Debian**: 使用 apt 自动安装
- **CentOS/RHEL**: 使用 yum 自动安装
- **Fedora**: 使用 dnf 自动安装
- 其他发行版：使用 nvm 安装

### macOS
- 使用 **Homebrew** 自动安装
- 如果没有 Homebrew，会自动安装

## 访问地址

启动成功后，浏览器会自动打开以下地址：

- **前端页面**: http://localhost:8080
- **后端API**: http://localhost:8899

### 局域网访问
如果想让同一局域网的其他设备访问：
1. 查看本机IP地址
   - Windows: `ipconfig`
   - Linux/Mac: `ifconfig` 或 `ip addr`
2. 在其他设备浏览器访问：`http://你的IP:8080`

## 常见问题

### Q: 双击后闪退怎么办？
**A**:
- Windows: 右键选择"以管理员身份运行"
- 查看是否有杀毒软件拦截

### Q: 浏览器打开后显示"无法访问"？
**A**:
- 等待前端编译完成（查看命令行窗口）
- 看到 "Compiled successfully" 后刷新页面

### Q: 提示端口被占用？
**A**:
- 先运行停止脚本（stop.bat 或 stop.sh）
- 然后重新运行启动脚本

### Q: Node.js 安装失败？
**A**:
- 访问 https://nodejs.org/ 手动下载安装
- 选择 LTS（长期支持）版本
- 安装完成后重新运行启动脚本
